## üöÄ Overview

**MediCheck** is a next-generation drug interaction checker reimagined for the AI era. Unlike legacy tools that rely on static tables, MediCheck uses **Google Gemini AI** to analyze complex multi-drug regimens, identifying subtle risks, mechanism-based conflicts, and clinical management strategies in real-time.

It features a **cyberpunk-inspired, clinical-grade UI** designed for speed, clarity, and visual impact.

## ‚ú® Features

### üß† AI-Core Analysis
*   **Deep Reasoning**: Goes beyond simple "major/minor" alerts to explain *why* drugs interact.
*   **Unlimited Regimens**: Check complex polypharmacy cases with 10+ medications simultaneously.
*   **Clinical Management**: Provides actionable mitigation strategies for healthcare professionals.

### üíé Interactive "Interaction Engine"
*   **Force-Directed Graph**: Visualize interaction clusters with a glowing, physics-based node graph.
*   **Severity Meters**: Dynamic radial gauges provide instant risk assessment at a glance.
*   **Spotlight Search**: Cinematic focus mode dims the interface for distraction-free entry.

### üõ†Ô∏è Utility & Personalization
*   **Sticky History**: One-click restore of recent searches from a persistent sidebar (Desktop).
*   **Lab Report Export**: Generate beautiful, patient-ready PDF reports with embedded graphs.
*   **Theme Engine**: Full Light/Dark/System mode support with "Neo-Clinical" aesthetics.

## üèÅ Getting Started

### Prerequisites
*   Node.js 18+
*   npm or pnpm

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/MedicoMrityunjay/medicheck.git

# 2. Enter the directory
cd medicheck

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```

Visit `http://localhost:8080` to launch the engine.
